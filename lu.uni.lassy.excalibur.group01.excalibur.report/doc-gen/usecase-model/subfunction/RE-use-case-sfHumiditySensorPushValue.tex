\subsubsection{subfunction-sfHumiditySensorPushValue}

\label{RE-use-case-sfHumiditySensorPushValue}


The humidity sensor has the ability to send the humidity level from the environment to the system. The system 
will update the value for the given sensor inside the sensor List.
