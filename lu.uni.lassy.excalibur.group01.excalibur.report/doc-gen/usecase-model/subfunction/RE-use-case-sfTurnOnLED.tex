\subsubsection{subfunction-sfTurnOnLED}

\label{RE-use-case-sfTurnOnLED}


In case the luminosity value is less than what the plants in a given room need and if there time ticker
is 0 the system will turn on the LEDs. Until the the ticker is 0 or the Luminosity value from the sensor has the value as the
expected one in the room.
