\subsubsection{usergoal-ugAddANewSensor}

\label{RE-use-case-ugAddANewSensor}


The add a new Sensor can only be executed if a request for a sensor has already be done and accepted by the
manager in addition the sensor request needs to be on true to successfully add a new sensor to the sensor list.
Which will be the result if you enter the correct values in the text fields inputs.
